Control of the Chemical Reactor

AC Motor user control simulates the process model of a chemical reaction tank, including the controlled systems for temperature and feeding level.
In combination with the PC’s (Programmable Controller) control functions it is possible to solve various training exercises of control engineering.
The reaction vessel is equipped with a double-walled jacket. A heating and cooling medium can be pumped through the interstice of the vessel. Inside the vessel two sensors are installed (for the present product temperature and the present feeding level) and a paddle mixer.

PROCESS DESCRIPTION

Our example process can be divided into 4 phases:

  • Feeding the vessel.
    During this phase the vessel is being fed with two products via two binary valves: V1 and V2.
  • Heating and stabilizing the product temperature.
    The analog valve AV1 is used as the actuator in order to continuously adjust the inflow quantity of the heating medium. A PI controller has been used. At the beginning of heating the product temperature increases linearly until it reaches the set value (this is obtained by linear increasing the set value for PI controller). Then the temperature is stabilized for 3 minutes. When this phase begins, the starting product temperature is stored in a TEM_REG register.
  • Cooling the product.
    The binary valve V3 opens the inflow of the cooling medium. The V4 valve must be opened too. This phase ends when product temperature reaches the value previously stored in TEM_PAM register.
  • draining off the vesse
    The drainage quantity of final product is continuously adjustable via an analog control signal AV1. In this exercise the AV1 valve is treated as a binary valve.

CONTROL PANEL FUNCTIONS

There are 3 process control modes:

  • Manual.
    Switches from 1 to 4 can be used to start the appropriate phase.
  • Auto.
    6th switch must be on. Then all the four phases are executed continuously.
  • One cycle.
    5th switch must be on. Then all the four phases are executed only once.
